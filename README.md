# AI-Powered HRM & Attendance System

A modern Human Resource Management (HRM) and Attendance System built with FastAPI, React.js, PostgreSQL, and AI-powered face recognition technology.

## Features

* Face Recognition Attendance
* Employee Management
* Leave Management
* Payroll Processing
* Shift Management
* Production Tracking
* Attendance Reports
* Dashboard Analytics
* PostgreSQL Database
* REST API Integration

## Technology Stack

### Backend

* Python
* FastAPI
* SQLAlchemy
* PostgreSQL

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3

### AI & Computer Vision

* OpenCV
* Face Recognition

git clone https://github.com/ar-sir23/hrm-face-attendance.git
